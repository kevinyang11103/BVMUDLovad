# 超市管理系统 java154

## 项目概述

本项目是基于Java技术的超市管理系统，采用SpringBoot和Vue.js实现前后端分离的架构设计。系统以B/S结构、MVC模式进行开发，提供完整的源码以及详细的文档和部署教程视频。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**: IDEA、JDK1.8、Maven、MySQL
- **技术栈**: Java、MySQL、SpringBoot、MyBatis-Plus、Redis、Vue.js

## 功能模块

### 用户模块
- 登录与权限验证

### 销售管理
- 销售主页
  - 购物结账
  - 积分兑换
  - 销售记录
  - 兑换记录
- 销售记录管理
- 积分兑换记录管理

### 人事管理
- 部门管理
- 员工管理

### 库存管理
- 仓库管理
- 出库明细
- 入库明细
- 供货商信息
- 库存统计

### 会员管理
- 会员信息管理

### 系统管理
- 角色管理
- 菜单管理

### 商品管理
- 分类管理
- 商品信息管理
- 积分商品
- 销售统计
